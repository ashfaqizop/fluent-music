# Fluent Music

A premium, native-feeling, high-performance, open-source **YouTube Music client for Windows**, built in Flutter, talking directly to YouTube's private InnerTube API. Music-only, Windows-only, ultra-polished custom Fluent UI.

Built by `ashfaqizop`. See [`Masterdoc.md`](Masterdoc.md) for the full specification and phased roadmap — it is the authoritative source of truth for this project.

## Status

Under active phased development (see `docs/phases.md`). Not yet ready for general use.

## Disclaimer

Fluent Music is an independent, unofficial project for educational and personal use. It is **not affiliated with, endorsed by, or sponsored by YouTube, YouTube Music, or Google**. It talks to YouTube's private InnerTube API, which is unofficial, reverse-engineered, and may break without notice. Users are responsible for complying with YouTube's Terms of Service.
